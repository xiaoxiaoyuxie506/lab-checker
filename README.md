# 实验室正常值范围检查系统

一个用于检查实验室检验项目正常值范围数据完整性和准确性的Web应用程序。

## 功能特点

- 📄 **上传Word文档**：支持上传.docx格式的表格数据
- 🔍 **自动检查**：
  - 性别完整性检查（每个项目需有男性和女性数据）
  - 年龄范围检查（下限 ≤ 上限）
  - 参考值范围检查（下限 < 上限）
  - 单位正确性检查（根据项目类型自动判断）
- 📊 **结果展示**：清晰的统计信息和错误详情
- 📥 **导出功能**：支持导出CSV、HTML报告和标记后的Word文档

## 技术栈

- **后端**：Python Flask
- **前端**：HTML + JavaScript + Bootstrap 5
- **文档处理**：zipfile + ElementTree（解析Word文档）

## 安装步骤

1. 克隆或下载项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行应用：
   ```bash
   python app.py
   ```
4. 打开浏览器访问：http://localhost:5000

## 使用说明

### 1. 准备数据

创建一个Word文档（.docx），包含以下列的表格：
- 项目名称
- 性别（男/女）
- 年龄下限
- 年龄上限
- 参考值下限
- 参考值上限
- 单位

### 2. 上传检查

1. 点击"选择文件"按钮或拖拽文件到上传区域
2. 点击"开始检查"按钮
3. 等待分析完成

### 3. 查看结果

- **统计卡片**：显示总记录数、错误数、警告数等
- **结果表格**：逐行显示检查结果
- **错误详情**：列出所有发现的问题

### 4. 导出结果

- **导出CSV**：导出为Excel可打开的CSV文件
- **导出HTML报告**：生成美观的HTML格式报告
- **导出标记文档**：导出原始Word文档（带标记）

## 检查规则

### 1. 性别完整性
每个检验项目必须包含男性和女性的数据。

### 2. 年龄范围
年龄下限必须小于或等于年龄上限。

### 3. 参考值范围
参考值下限必须小于参考值上限。

### 4. 单位正确性

| 项目类型 | 正确单位 |
|---------|---------|
| 血细胞计数类（白细胞、红细胞、血小板等） | 10^9/L 或 10^12/L |
| 血红蛋白/蛋白类 | g/L |
| 胆红素/肌酐/尿酸 | μmol/L |
| 尿素类 | mmol/L |
| 电解质（钾钠氯钙磷镁） | mmol/L |
| 酶类（AST、ALT、ALP、GGT等） | U/L |
